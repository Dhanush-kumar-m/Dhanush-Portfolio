## 💻 Local Development Setup

To run this portfolio locally:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Dhanush-kumar-m/Dhanush-Portfolio.git
    cd Dhanush-Portfolio
    ```

2.  **Install dependencies:**
    This project is lightweight and uses `package.json` configurations.
    ```bash
    npm install
    ```

3.  **Start the local development server:**
    ```bash
    npm run dev
    ```
    This will spin up a local HTTP server and open the portfolio automatically in your web browser.
